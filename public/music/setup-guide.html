⚠️ File Musik Diperlukan
Anda perlu menambahkan file musik dengan nama background-music.mp3 ke folder ini.

🎼 Langkah Setup Musik:

  1. Download musik instrumental/ambient dari sumber legal
  2. Rename file menjadi background-music.mp3
  3. Letakkan di folder public/music/
  4. Refresh halaman thread untuk mendengar musik
💡 Rekomendasi:
• Gunakan musik instrumental yang tenang
• Ukuran file maksimal 5MB
• Durasi 3-10 menit (akan loop otomatis)
• Format MP3 untuk kompatibilitas terbaik

🎛️ Fitur Music Player:

  • ✅ Auto-play (sesuai kebijakan browser)
  • ✅ Kontrol play/pause
  • ✅ Kontrol volume
  • ✅ Tombol close untuk menyembunyikan
  • ✅ State tersimpan antar halaman
  • ✅ Desain responsive
